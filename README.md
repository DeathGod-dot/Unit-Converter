# 📐 Unit Converter - Modern Measurement Toolkit

A sleek, modern Android application designed to make everyday unit conversions effortless. **Unit Converter** delivers instant, accurate conversions through an elegant Material 3 interface built entirely with **Jetpack Compose**.

---

## ✨ Key Features

### 📏 **Comprehensive Unit Conversion**
* **Multiple Categories**: Convert across various measurement types from a single interface.
* **Instant Results**: Live conversion as you type.
* **High Precision**: Accurate calculations using standardized conversion factors.
* **Bidirectional Conversion**: Quickly swap between input and output units.

### 🎨 **Modern User Experience**
* **Material 3 Design**: Clean, responsive interface following Google's latest design language.
* **Jetpack Compose UI**: Smooth animations and a fully declarative UI.
* **Minimal Navigation**: Designed for quick access with minimal taps.
* **Responsive Layout**: Optimized for different Android screen sizes.

### ⚡ **Performance**
* **Instant Calculations**: No delays while converting values.
* **Lightweight Architecture**: Fast startup and low memory usage.
* **Offline Support**: No internet connection required.
* **Efficient State Management**: Smooth user interactions powered by Compose.

### 🛠 **Reliable Conversion Engine**
* **Standardized Conversion Factors**
* **Decimal Precision Support**
* **Input Validation**
* **Real-time Error Handling**
